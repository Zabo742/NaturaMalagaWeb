# Natura Málaga - Head Spa Premium

Sitio web para Salón de Belleza Natura, especializado en Head Spa Premium y servicios de bienestar en Málaga.

## Características

- Reserva de citas online
- Catálogo de servicios
- Galería de tratamientos
- Información de contacto

## Tecnologías

- React 18 con TypeScript
- Vite como bundler
- Tailwind CSS para estilos
- Radix UI para componentes accesibles
